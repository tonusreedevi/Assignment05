1- What is the difference between var, let, and const?
var is function scoped and can be redeclared, while let is block-scoped and cannot be redeclared in the same scope.
const is also block scoped but its value cannot be reassigned after declaration const is fixed for the whole document .

2- What is the spread operator (...)?

The spread operator is used to expand elements of an array or object into individual values.
It is commonly used to copy arrays, merge arrays, or pass multiple values into functions.

3️- What is the difference between map(), filter(), and forEach()?

map() returns a new array by transforming each element in map you pass a function that execute for each value of array, while filter()  returns a new array with elements that pass a condition its also like map().

forEach() only loops through elements and performs an action but is also similar to map but it returns different value

4- What is an arrow function?


An arrow function is a shorter way to write functions using the => syntax.

It does not have its own this and is often used for simple or callback functions. it is a ES6 features.
